# Front End Test

MercadoLibre Front End Test

## Getting Started

### Prerequisites

Install:
- nodeJS
- Ruby & Sass
- bower

### Running

1. Run:
```
npm install
```
2. Run:
```
grunt
```
3. Run:
```
npm start
```

Express JS server will be started at:

localhost:3000
